# 🥕 당근마켓 클론코딩

## 💬 프로젝트 개요 

공식 당근마켓에는 앱과 달리 PC페이지에는 로그인 기능이 없습니다.

이 프로젝트는 웹페이지에서 로그인을 만들어 모바일 앱처럼 이용해보자는 취지로 시작하였습니다.

더불어 기간이 짧은만큼 "중고거래 플랫폼 당근마켓 만들기를 통한 스프링 익숙해지기"에 목적을 두었습니다.
